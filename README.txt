 Описание:

 Проект выводит сумму заказов города выбранного в ListBox, через textBox можно фильтровать имена городов по имени.


 Реализовано:

 ИНтерфейс

 Button Order

 Вывод городов в ListBox

 Суммы сделанных заказов в городе

 Показ заказов в datagridview


 Не Реализовано:

 Button для очистки 


 Что использовано:

 SQLite

 customers.db который мы создали от импортированных .csv файлов
